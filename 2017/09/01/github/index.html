Github

Sep 01, 2017效率 | 工具点击

一、注册 github 账号、创建个人的 github 仓库(略过)

二、配置 SSH Keys

①、终端输入命令使隐藏文件可见：

defaults write com.apple.finder AppleShowAllFiles -bool true

②、重启 finder：左上角苹果图标 -> 强制退出 -> finder -> 重新开启

③、前往个人文件查看是否有 .ssh 文件夹：前往 -> 个人。如果有，建议删除后重新配置

④、创建一个 .ssh 文件夹：mkdir .ssh

⑤、进入刚创建的 .ssh 文件夹：cd .ssh 。输入命令：ssh-Keygen -t rsa -C “注册时的邮箱 YourEmail”

⑥、输入完成后一直按回车，中间会提示要输入密码也不用管，直到如下图：

⑦、输入命令：ls -la 。出现如下信息则说明配置成功：

⑧、输入命令：pbcopy < ~/.ssh/id_rsa.pub 复制到剪切板

⑨、登录 github 进入 SSH Keys：Settings -> SSH and GPG keys -> New SSH key 。标题随意填写，key 粘贴刚才复制的内容，添加后如下：

⑩、终端输入命令：ssh -T git@github.com . 终端显示 Are you sure you want to continue connecting (yes/no)? 输入 yes 回车，返回 github 页面，如下所示代表成功：

三、上传代码到 github

①、下载 github 客户端并登录账号

②、clone 刚创建的仓库到本地

③、打开刚 clone 到本地的文件夹，将需要上传的内容放到文件夹内

④、回到 github 客户端，选中 clone下来的仓库，可以看见新增了文件且需要提交，填写好 “summary”、”description”，然后点击 commit to master，然后点击右上角按钮

⑤、在github 网站查看是否提交成功
