## 1、github clone 提速

1. 在 [IPAddress.com](https://www.ipaddress.com/) 网站中中分别查询 ``https://www.github.com`` 和 ``github.global.ssl.fastly.net`` 域名的 IP 地址。

2. 打开的 hosts 文件

【Windows】

```
C:\Windows\System32\drivers\etc\host
```

【Mac】

```
$ sudo vi /etc/hosts
```
