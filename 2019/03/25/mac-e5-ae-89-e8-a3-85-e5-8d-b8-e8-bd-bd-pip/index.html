检测是否已安装 pip

~ $ pip
-bash: pip: command not found

如果出现 -bash: /usr/local/bin/pip: No such file or directory 或如上的提示，说明还未安装 pip。

查看 pip 版本

~ $ pip –version
pip 19.0.3 from /Library/Python/2.7/site-packages/pip-19.0.3-py2.7.egg/pip (python 2.7)

卸载 pip

只需要一个命令 sudo pip uninstall pip 然后会提示你是否确认卸载，输入 y 就可以了。

~ $ sudo pip uninstall pip
Password:

Successfully uninstalled pip-19.0.3

最后出现 Successfully uninstalled 的字样就说明卸载完成。
